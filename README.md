# LegalEase

**Simplifying law with machine learning.**

## Overview
LegalEase is an AI-powered project that takes complex legal documents—contracts, policies, agreements—and translates them into clear, plain language. The goal is to bridge the gap between legal jargon and everyday understanding, making legal text accessible to non-experts without losing accuracy.

## Features
- ⚖️ **Clause Simplification** – Breaks down dense legal clauses into digestible explanations.  
- 🤖 **Machine Learning Models** – Uses NLP and ML techniques to detect, classify, and simplify legal text.  
- 📖 **Plain Language Output** – Generates summaries and explanations in everyday language.  
- 🔍 **Context Preservation** – Ensures meaning and intent remain intact while simplifying.  

## Why It Matters
Legal documents are notoriously difficult to read. By leveraging machine learning, LegalEase empowers individuals and businesses to understand their rights and obligations without needing a law degree.

## Getting Started
1. Clone the repository:  
   ```bash
   git clone https://github.com/chiragthedefender/LegalEase.git
   ```
2. Install dependencies:  
   ```bash
   pip install -r requirements.txt
   ```
3. Run the simplifier:  
   ```bash
   python main.py --input contract.pdf --output simplified.txt
   ```
